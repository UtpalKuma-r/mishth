Making A Spam Bot

Its fun to troll our friends. And in this world of social media its to good to spam our friends account with plenty of messages.

For this great pleasure I decided to make something which can help me with this.

The following python code autometically types and sends a message to the specified account.

import pyautogui, time   
num = int(input("Enter the number of msg you want to send: "))
write = input("what do you want to spam: ")
time.sleep(4)   #we get 4 seconds before the bot starts typing.
                  It is a good idea to open the the social media 
                  page before this time runs out.
while num>0:
    pyautogui.typewrite(write)   #types the message
    pyautogui.press("enter")   #presses enter to send the message
    num = num-1
    time.sleep(2)
    Steps to Make it work
  • Open your Browser
  • Login in your account
  • Run the code
  • Place the courser in the writing area
  • Sit back and relax
